Name

vfs_readonly — 指定された時間、Samba共有を読み込み専用にする

Synopsis

vfs objects = readonly

説明

このVFSモジュールは samba(7)システムの一部である。

vfs_readonly VFSモジュールは指定された時間、 共有を読み込み専用とする。この間のクライアントからの接続の書き込み アクセスは、アクセス権限によらず、すべて拒否される。

このモジュールはスタッカブルである。

オプション

readonly:period = BEGIN, END

クライアントからの接続が、BEGINとENDで指定された間、共有は読み込み専用となる。 日付を指定する構文はGNU date(1)の-dオプションと同じ。

使用例

すべての共有を読み込み専用とする:

        [global]	vfs objects = readonly

[backup]共有を、業務時間の間読み込み専用とする

        [backup]	path = /readonly	vfs objects = readonly	readonly:period = readonly:period = "today 9:00","today 17:00"

バージョン

このマニュアルはSambaシステムバージョン3.0.25(訳注:3.3?)に対応している
