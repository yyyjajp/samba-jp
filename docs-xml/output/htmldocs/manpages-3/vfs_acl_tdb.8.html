Name

vfs_acl_tdb — tdbファイルにNTFSのACLを保存する

Synopsis

vfs objects = acl_tdb

説明

このVFSモジュールは samba(7)システムの一部である。

vfs_acl_tdb VFSモジュールは、 tdbファイルにNTFSのアクセスコントロールリスト（ACL)を格納する。 これは、Sambaサーバ上でWindows ACLの完全なマッピングを有効にする。

ACLの設定は $LOCKDIR/file_ntacls.tdb中に格納される。

このモジュールは 実験的であることに留意して欲しい!

このもジュールはスタッカブルである。

オプション

vfs_acl_tdbに対してオプションはない。
