Name

vfs_audit — 選択したSamba VFSモジュールの動作状況をsystem logに保存する

Synopsis

vfs objects = audit

説明

このVFSモジュールは samba(7)システムの一部である.

vfs_audit VFSモジュールは指定したクライアントの動作を syslog(3)を使用して、システムログに記録する。

以下のようなSamba VFSの動作が記録される

connect
disconnect
opendir
mkdir
rmdir
open
close
rename
unlink
chmod
fchmod
chmod_acl
fchmod_acl

このモジュールはスタッカブルである

オプション

audit:facility = FACILITY

syslog(3) にFACILITYでメッセージを記録する。

audit:priority = PRIORITY

syslog(3)にPRIORITYでメッセージを記録する。

使用例

すべての共有のログをLOCAL1ファシリティー、NOTICEプライオリティーで記録する。 

        [global]
	vfs objects = audit
	audit:facility = LOCAL1
	audit:priority = NOTICE

バージョン

このマニュアルはSambaシステムバージョン3.0.25に対応している
